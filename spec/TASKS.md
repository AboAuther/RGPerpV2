# 任务

## 使用说明

- 本文档按里程碑推进，每完成一项必须打勾。
- 任何涉及资金、账本、风控、清算的实现，必须同时满足：资金安全优先于一切、风控优先于交易、所有数据必须可追溯、所有操作必须可审计。
- 若实现与以下规范冲突，以 `docs/` 与 `spec/` 中最新实施级文档为准。
